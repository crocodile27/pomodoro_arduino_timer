# Pomodoro Arduino Timer
Physical Hardware Timer using an 8*8 Neomatrix, button and potentiometer.

The arduino code is located in this repo. 

### Device Specs:
- Arduino ELEGOO UNO R3
- Adafruit Neomatrix
- B10 K Potentiometer
- Two legged button
- 3.3 Ω Resistor
- Standard Wires
